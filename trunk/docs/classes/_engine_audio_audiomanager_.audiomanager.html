Class AudioManager

Raykid

email

initial_r@qq.com

create

date 2017-10-30

modify

date 2017-10-30

音频管理器，音频接口被强行分为两部分：Sound和Music。 Sound：使用Audio标签播放，可以跨域播放但可能会被某些浏览器限制，必须在点击事件处理函数中播放 Music：使用AudioContext播放，可以一定程度上越过点击事件检查，但无法跨域播放，适合播放背景音乐
